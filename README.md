# Cyber Video Player

## Default Keybindings

| Key | Description |
| --- | ----------- |
| Space | Play/Pause |
| F | Toggle fullscreen |
| Ctrl+Left | Frame step backward |
| Ctrl+Right | Frame step forward |
| M | Mute |
| Ctrl+T | Switch to trim control |
| I | Set the start time for trimming |
| O | Set the end time for trimming |
